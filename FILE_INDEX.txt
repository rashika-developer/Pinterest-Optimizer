================================================================================
TROUBLESHOOTING BY FILE
================================================================================

Issue: Build errors
→ Check: build.gradle dependencies
→ Fix: Run "Sync Now"

Issue: "Cannot resolve symbol"
→ Check: File location matches package name
→ Fix: Move file to correct folder

Issue: App crashes on launch
→ Check: MainActivity, AndroidManifest
→ Fix: Check Logcat for exact error

Issue: API calls fail
→ Check: api.kt, di_module.kt
→ Fix: Verify backend URL is correct

Issue: Data not saving
→ Check: database.kt, dao.kt
→ Fix: Check @Entity and @Query annotations

Issue: UI doesn't show
→ Check: *Screen.kt files
→ Fix: Verify Compose dependencies installed

Issue: Navigation broken
→ Check: MainActivity.kt NavHost
→ Fix: Verify all routes match screen names
